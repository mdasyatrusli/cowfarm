# Frontend Rules

## Purpose

Standar UI menggunakan Blade dan Tailwind.

---

## Technology

- Blade
- Tailwind CSS
- Alpine.js (optional)

---

## Layout

Gunakan:

layouts/

components/

pages/

---

## Component

Selalu gunakan:

x-button

x-card

x-modal

x-table

---

## Responsive

Mobile First.

---

## UX

Form harus memiliki:

- Label
- Validation
- Placeholder

---

## Tables

Gunakan:

- Search
- Pagination
- Sorting

---

## Never

- Inline CSS
- Inline JavaScript
- Duplicate component
