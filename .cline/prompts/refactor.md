# Refactor Prompt

## Purpose

Membantu AI meningkatkan kualitas kode tanpa mengubah perilaku aplikasi.

---

## When to Use

- Controller terlalu besar
- Duplicate code
- Function terlalu panjang
- Query terlalu kompleks
- Naming buruk

---

## AI Thinking Process

Analisis

↓

Identifikasi Code Smell

↓

Prioritas Refactor

↓

Implementasi

↓

Regression Check

---

## Code Smells

- Duplicate Code

- Long Method

- Long Controller

- Complex If Else

- Magic Number

- Tight Coupling

---

## Refactoring Rules

Tidak boleh:

- mengubah business logic
- mengubah database tanpa alasan
- menghapus fitur

---

## Output

Current Problem

↓

Improvement Plan

↓

Refactored Code

↓

Benefits

---

☐ Lebih mudah dibaca
