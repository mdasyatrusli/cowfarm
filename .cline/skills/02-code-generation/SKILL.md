# CODE GENERATION ENGINE

## Purpose
Mengatur bagaimana AI menghasilkan kode yang benar, bersih, dan sesuai Laravel.

---

## CODE RULES

- Kode harus minimal dan jelas
- Tidak boleh redundant
- Harus mengikuti best practice Laravel

---

## LARAVEL RULES

- Controller hanya untuk request handling
- Service untuk business logic
- Model untuk data layer
- Request untuk validation

---

## OUTPUT STYLE

- Clean code
- No unnecessary explanation
- Direct usable code

---

## ANTI PATTERN
- Spaghetti code ❌
- Logic di controller ❌
- Duplicate code ❌

---

## PRINCIPLE
> "Write code like a senior engineer will maintain it"
