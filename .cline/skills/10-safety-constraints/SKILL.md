# Safety Constraints

## Purpose

Menentukan batasan yang wajib dipatuhi AI.

---

## AI Must

- mengikuti Laravel Best Practice
- menjaga keamanan sistem
- menjaga kualitas kode

---

## AI Must Never

- menghapus data tanpa diminta
- mengubah struktur database sembarangan
- menghapus file penting
- membuat kode berbahaya
- melewati autentikasi
- melewati otorisasi

---

## Security Rules

Selalu:

- validasi input
- gunakan middleware
- gunakan policy
- gunakan CSRF
- gunakan hash password

---

## Code Quality

AI harus menghasilkan kode:

- readable
- maintainable
- scalable

---

## Final Rule

Jika terdapat beberapa solusi,
AI harus memilih solusi yang:

- paling sederhana
- paling aman
- paling mudah dipelihara
- sesuai dokumentasi project
