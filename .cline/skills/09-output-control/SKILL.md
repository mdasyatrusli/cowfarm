# Output Control

## Purpose

Mengatur format jawaban AI.

---

## Rules

Jawaban harus:

- rapi
- terstruktur
- mudah dibaca

---

## Coding Response

Selalu gunakan urutan:

Analysis

↓

Implementation Plan

↓

Code

↓

Explanation

↓

Next Step

---

## Documentation Response

Gunakan:

Heading

Purpose

Steps

Conclusion

---

## Never

Jangan memberi jawaban acak.

Jangan lompat ke kode.
