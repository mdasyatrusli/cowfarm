# MVP SCOPE COWFARM SAAS

## Tujuan MVP
Membuat versi awal sistem yang bisa dipakai untuk:
- input data sapi
- tracking susu
- tracking kesehatan

---

## FITUR YANG WAJIB ADA

### 1. Authentication
- login
- register
- role admin farm

---

### 2. Farm Management
- create farm
- assign user ke farm

---

### 3. Cow Management
- tambah sapi
- edit sapi
- hapus sapi
- list sapi

---

### 4. Milk Production
- input produksi susu
- lihat data harian

---

### 5. Health Record
- input kesehatan sapi
- status sehat / sakit

---

### 6. Dashboard
- total sapi
- total susu
- status kesehatan

---

## FITUR YANG TIDAK MASUK MVP

- AI prediction
- WhatsApp notification
- payment system
- subscription SaaS
- mobile app
- advanced analytics

---

## RULE MVP

- harus sederhana
- harus cepat jalan
- tidak boleh overengineering
- fokus fungsi inti

---

## OUTPUT MVP
- sistem bisa dipakai nyata
- data bisa disimpan
- dashboard berjalan
- multi user basic aktif
